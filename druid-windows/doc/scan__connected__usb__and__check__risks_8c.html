Scans connected USB devices and checks for potential risks.

Loads the USB database from file, iterates through connected USB devices, compares each device against the database, and updates risk statistics.

int scan_connected_usb_and_check_risks( usb_tools_t *usb_tools, usb_device_info_t *usb_device_info, usb_db_entry_t *usb_db_entry, cli_args_t *cli_args)

Parameters
usb_toolsPointer to the usb_tools_t structure used for device enumeration
usb_device_infoPointer to the usb_device_info_t structure for storing device info
usb_db_entryPointer to the usb_db_entry_t structure for matching database entries
cli_argsPointer to the cli_args_t structure containing CLI arguments
Returns
Exit code:
  • 0 (EXIT_SUCCESS) when scanning and risk checking complete
  • 84 (EXIT_ERROR) if database loading fails
